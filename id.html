Services

Production of interactive 3d content, displayable on web pages, for technical, promotional or other purposes. The produced content allows users to have an experience very similar to real life manipulation of the objects. The interaction width the 3d objects in the 3d space makes possible to interact with the objects at distance via the internet and explore all the features of the products. The page with the 3d content can be open by any modern internet browser in all kind of devices with gpu support, like mobile phones, tablets, touchscreens and pc's .

Creation of 3d models, vectorial maps, textures, normal maps, light maps, bump maps, vectorial graphics and other graphic design content. Integration of 3d content in web pages, providing motion and interaction with the user when required.

Environment creation. Natural or conceptual environments where the models will be displayed. A more realistic perception is created using an environment. Details like gloss or reflection can be enhanced by the surrounding environment. The user can change his point of view inside the environment, explore the model from diferent angles and zoom in at details of his interest.
