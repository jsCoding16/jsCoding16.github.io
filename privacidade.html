Política de Privacidade

Esta Política de Privacidade descreve como a Mynav recolhe, utiliza e partilha as suas informações ao utilizar a nossa aplicação.

1. Dados que Recolhemos

  • Dados Pessoias: A MyNav não recolhe dados pessoais (nome, email, contactos).
  • Dados recolhidos automaticamente: Identificação Técnica - Utilizamos o Android ID exclusivamente para validar a licença de acesso aos ficheiros de Roadbook através do nosso servidor (Supabase/GitHub). Este ID não é partilhado com terceiros.
  • Dados de localização: Localização (GPS) - A aplicação processa a sua localização em tempo real apenas para mostrar a sua posição nos mapas offline (Mapsforge). Não armazenamos nem transmitimos o seu histórico de localização.

2. Segurança e Retenção

Utilizamos encriptação (security-crypto) para proteger as preferências locais no seu dispositivo. Implementamos medidas técnicas de segurança para proteger os seus dados contra acessos não autorizados. Manteremos as suas informações apenas pelo período necessário para cumprir as finalidades descritas, exceto se a lei exigir um período de retenção superior.

3. Os Seus Direitos

De acordo com a legislação de proteção de dados (como o RGPD ou LGPD), o utilizador tem o direito de:

  • Aceder, corrigir ou solicitar a eliminação dos seus dados pessoais;
  • Revogar o consentimento para o tratamento de dados a qualquer momento;
  • Solicitar a portabilidade dos seus dados para outro serviço.
